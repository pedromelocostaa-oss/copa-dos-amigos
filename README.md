# Copa dos Amigos

Projeto colaborativo entre amigos para organizar a Copa dos Amigos.

## Como participar

1. Crie uma conta no [GitHub](https://github.com)
2. Peça para ser adicionado como colaborador do repositório
3. Clone o projeto no seu computador:
   ```bash
   git clone https://github.com/pedromelocostaa-oss/copa-dos-amigos.git
   ```
4. Crie uma branch com seu nome antes de fazer alterações:
   ```bash
   git checkout -b seu-nome/o-que-voce-fez
   ```
5. Faça suas alterações, salve e envie:
   ```bash
   git add .
   git commit -m "Descrição do que você fez"
   git push origin seu-nome/o-que-voce-fez
   ```
6. Abra um Pull Request no GitHub para revisar antes de juntar ao projeto principal

## Estrutura do projeto

```
copa-dos-amigos/

├── dados/       # Planilhas e dados do campeonato
└── codigo/      # Scripts e código
```
